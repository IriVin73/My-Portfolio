Recipe App

A feature-rich recipe app with a user authentication system, enabling personalized experiences. The app fetches recipes from an API and displays them in a randomly selected, scrollable list. Users can filter recipes by country, meal type (e.g., breakfast, appetizers, drinks), and search by keywords, allergies, preparation time, cuisine, and diet preferences. The recipe form allows for adjustable serving sizes, which dynamically update ingredient measurements. Nutritional information is displayed for each recipe, and users can add their favorite recipes to cookbooks or planners. Additionally, nutritional intake is visualized with interactive graphs, providing insightful tracking of your food choices.

Drinks Machine

This interactive application allows users to choose from three different drink recipes. Once a drink is selected, the program guides the user through the preparation process step by step.

Each step is presented individually, accompanied by:

  • A corresponding image to visually demonstrate what to do
  • A text description explaining the action
  • An optional text-to-speech feature that reads the instructions aloud

Users can easily navigate between steps using Next and Previous buttons, allowing them to move at their own pace.

When all steps are completed, the program displays a summary screen listing all the ingredients required for the chosen drink, making it easy to review or prepare for next time.

This application offers a smooth and engaging experience for users of all ages to follow along with simple drink recipes in a fun and interactive way.
